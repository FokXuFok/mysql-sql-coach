# SQL Coach 🤖

> AI-powered MySQL slow query optimization coach

SQL Coach is a CLI tool that analyzes MySQL slow queries using AI. Input a SQL statement, and it connects to MySQL to run EXPLAIN, calls an AI model to analyze the execution plan, outputs optimization suggestions, rewrites the SQL, generates index DDLs, runs performance benchmarks, and lets you choose which SQL version to use (copied to clipboard).
